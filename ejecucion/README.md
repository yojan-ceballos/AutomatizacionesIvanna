# Ejecución: Scripts Deterministas

Este módulo contiene scripts Python que ejecutan tareas de forma confiable y determinista.

## Principios

1. **Determinísticos** - Mismo input = mismo output
2. **Bien comentados** - Fáciles de entender y mantener
3. **Testeables** - Pueden probarse de forma aislada
4. **Auto-contenidos** - Variables de entorno en `.env`
