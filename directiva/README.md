# Directivas (POEs)

Esta carpeta contiene los Procedimientos Operativos Estándar (POEs) en formato Markdown.

## ¿Qué es una directiva?

Una directiva define **qué hacer**, no cómo hacerlo técnicamente. Es como las instrucciones que darías a un empleado de nivel medio.

## Estructura de una directiva

```markdown
# Nombre del Proceso

## Objetivo
Qué se busca lograr.

## Entradas
- Datos, archivos o información necesaria

## Herramientas/Scripts
- Scripts de `ejecucion/` que se usan

## Salidas
- Resultados esperados

## Casos límite
- Qué hacer cuando algo falla
```

## Principios

1. **Lenguaje natural** - Escribe como si explicaras a un humano
2. **Actualizar con aprendizajes** - Documenta errores encontrados y soluciones
3. **Referencia scripts existentes** - Revisa `ejecucion/` antes de crear nuevos
